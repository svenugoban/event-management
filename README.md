# Event Management

## Overview

This repository contains the code and resources for the 'Event Management' project.

## Features

## Technologies Used

- **Backend:** Node.js
- **Frontend:** React
- **Database:** MySql

Project Setup

# Front End 
1. clone the project from github
2. inside the project folder run "npm install"
3. run "npm start"


# Back End 
1. clone the project from github
2. inside the project folder run "npm install"
3. run "npm run dev"
